Step 1

  • Go to /admin with ?password= and copy admin_token
    • admin password
  • Go to /create-user with ?token='INSERT admin_token', and copy user auth token

Step 2

?token= IS REQUIRED PARAM FOR EACH REQUEST

  • /best/ - get first students, ?count= for count, default=1
  • /last/ - get last students, ?count= for count, default=1
  • /random/ - get random students, ?count= for count, default=1
  • /place/ with required ?score= get place in table by total score
  • /find/ with one of required params ?place= or ?snils= for get student (place > snils)

Only with admin token

  • /update/ - update database, ?token=
  • /create-user/ - creating new user, ?token=
